------------------------------------------------------------------------
                               葵2 CGI環境
------------------------------------------------------------------------
* 概要
レンタルサーバー等に葵をインストールして利用することができます。
------------------------------------------------------------------------
* 必要な環境

-サーバー側     .. PHP4 + SQLite2 が動く環境
-クライアント側 .. Java (JRE) 1.4 以降 / Flash Player 9以降
------------------------------------------------------------------------
* セットアップ

サーバーに取得したファイルとディレクトリを全てコピーする。
PHP ファイルのパーミッションを 766 に設定する。

***(1) Webブラウザから、以下のURLにアクセスしてSQLiteデータベースを初期化します

cgi/setup.php

エラーが出た場合、多くは、パーミッションの設定によるものです。

|ファイル|パーミッション|
|*.php   |766           |
|*.js    |666           |
|*.xml   |666           |
|フォルダ|777           |


***(2) セットアップに使った不要なファイルを削除します。

/cgi/setup.php を削除
/cgi/setup フォルダを削除

***(3) エディタの設定ファイルを書き換えます

config_editor.xml の以下の id=base の行の value を 設置するサーバーの URL に書き換えます。

	<section id="path">
		<key id="base" value="http://localhost/aoi2/" />
	</section>

***(4) CGIのための設定ファイルを書き換えます

config_cgi.js の以下の行を設置するサーバーの URL に書き換えます。

 var baseUrl = "http://localhost/aoi2/";
